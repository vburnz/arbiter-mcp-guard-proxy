MCP tool-call firewall.

v0.5.0 Apache-2.0 Rust github.com/cyrenei/arbiter-mcp-firewall

Arbiter governs what agents are allowed to do. It does not govern what agents try to do. It sits between your AI agents and the MCP servers they talk to, enforcing deny-by-default authorization on every tool call.

Traditional access control was built for humans who log in and make decisions. Arbiter was built for agents that fire hundreds of tool calls per session. It adds session budgets, tool whitelisting, drift detection, credential injection, and structured audit logging with PII redaction.

Install

curl -sSf https://raw.githubusercontent.com/cyrenei/arbiter-mcp-firewall/main/install.sh | sh

Downloads the latest binary for your platform (Linux or macOS) with SHA256 verification. No sudo required.
